🟥 HIGH‑RISK (Secuencial estricto)
Estos instaladores NO pueden ejecutarse junto a ningún otro, porque:

Usan MSI directamente
Instalan drivers, servicios, antivirus, componentes del sistema
Requieren exclusividad total
Bloquean Windows Installer
Lanzan errores 1618 / 1223 si hay otro instalador activo

✔ Incluye:
Todos los .MSI
Office (Click‑to‑Run, O365, OfficeSetup.exe)
Sophos (Endpoint, Agent, Intercept X)
Antivirus / EDR (CrowdStrike, SentinelOne, CarbonBlack, etc.)
Drivers (impresoras, escáneres, USB, audio, video)
TSPrint / TSScan
TightVNC MSI
Spiceworks Agent
eRAD Desktop Apps
Instaladores corporativos pesados
Instaladores que requieren reinicio
Instaladores que modifican servicios del sistema

✔ Regla:
Siempre uno por uno. Nunca en paralelo.

🟧 MEDIUM‑RISK (Secuencial con espera inteligente)
Estos instaladores sí pueden ejecutarse, pero:

Usan MSI internamente
Lanzan procesos secundarios que bloquean otros instaladores
Requieren UAC
No son seguros para paralelismo

✔ Incluye:
ChromeSetup.exe
Firefox Installer
Adobe Reader / Acrobat
Java Runtime / JDK
7‑Zip EXE
WinRAR EXE
AnyDesk / TeamViewer
Zoom / Teams / Webex
Instaladores EXE genéricos que contienen “setup”, “installer”, “update”
Instaladores que registran DLLs o servicios

✔ Regla:
Uno por uno, pero solo esperar si otro instalador está activo.

🟩 LOW‑RISK (Paralelo seguro)
Estos instaladores:

No usan MSI
No instalan drivers
No modifican servicios
No requieren exclusividad
Son ligeros (NSIS, Inno Setup, InstallShield)
Solo copian archivos o instalan componentes simples

✔ Incluye:
Inno Setup installers(contienen “Inno Setup” dentro del EXE)
NSIS installers(contienen “Nullsoft Installer”)
InstallShield installers
Aplicaciones portables
ZIP autoextraíbles
Utilidades pequeñas
Herramientas sin servicios
Apps que no requieren UAC
Instaladores que no tocan el sistema

✔ Regla:
Se pueden ejecutar varios al mismo tiempo.
